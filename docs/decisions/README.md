# 架构决策记录 (DDR)

> 本目录存放项目的架构决策记录 (Design Decision Records)。

## 格式

```
DDR-NNN: <简短标题>

- 日: YYYY-MM-DD
- 状态: proposed | accepted | deprecated | superseded
- 背景: 为什么需要决策
- 决定: 做出了什么选择
- 影响: 对项目的影响
- 替代方案: 考虑过但未选择的其他方案
```
