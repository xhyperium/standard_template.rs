# 编码与语言约定

本仓库统一遵循以下规则。

> **宪章效力**：见 [docs/constitution/04-code-standards.md §4.5](../constitution/04-code-standards.md#45-语言与编码强制)。  
> **组织上位（不可削弱）**：[`xhyperium/.github` → `rulesets/language.md`](https://github.com/xhyperium/.github/blob/main/rulesets/language.md)。  
> 冲突时：**组织 language.md > 宪章 §4.5 > 本文件**。

## 字符编码

- **所有文本文件使用 UTF-8**（无 BOM）
- 换行符统一为 **LF**
- 由 `.editorconfig` 声明 `charset = utf-8`
- 禁止提交 GBK / GB2312 / UTF-16 等其他编码的源文件

## 语言（强制中文）

| 类别 | 语言 |
| ------ | ------ |
| 代码注释 | **中文** |
| 项目治理 / 协作文档（宪章、AGENTS、PR/Issue、设计规格） | **中文** |
| 用户可见错误信息（`Display` / 业务文案） | **中文** |
| Agent 对用户输出 / 审查 / handoff | **中文** |
| 提交信息 | Conventional Commits：`type(scope): 中文说明` |
| 标识符（类型名、函数名、模块名） | 英文（Rust 惯例） |
| 法律文本（`LICENSE`） | 英文（许可证原文） |
| 第三方技能 / 上游文档 | 可保留原文；**本组织新增默认中文** |
| 英文技术交付正文 | **非默认**；须书面豁免；若启用可参考 [ASD-STE100.md](./ASD-STE100.md)（§4.6 可选） |

**禁止**：无豁免的大段英文文档或英文 Agent 汇报作为默认交付。

## 技术术语

可保留英文术语本体，例如：API、Docker、Kubernetes、CI、PR、workspace、crate。  
解释性语句使用中文。

## 自检

```bash
# 粗检：非 UTF-8 文件（应无输出）
find . \( -path ./.git -o -path ./.cargo/target -o -path ./.beads \) -prune -o \
  -type f \( -name '*.rs' -o -name '*.md' -o -name '*.toml' -o -name '*.yml' \) -print \
  | while read f; do iconv -f UTF-8 -t UTF-8 "$f" >/dev/null 2>&1 || echo "NON-UTF8 $f"; done
```
