# docs/ — 文档索引

本目录按**文档职责**严格分类。新增文档必须先归类，禁止在 `docs/` 根目录平铺内容文件（索引除外）。

## 分类规则

| 目录 | 放什么 | 不放什么 |
|------|--------|----------|
| [`constitution/`](constitution/) | 工程宪章正文 SSOT | 落地细则、状态报告 |
| [`governance/`](governance/) | 宪章落地细则、强制工程约定 | 一次性状态报告 |
| [`standards/`](standards/) | 编码标准、领域规范 | 宪章正文 |
| [`decisions/`](decisions/) | 架构决策记录（DDR） | 日常状态报告 |
| [`status/`](status/) | CI/仓库配置的状态与验证记录（可过期） | 长期有效的规则 |
| [`report/`](report/) | 只读审计/就绪度报告（可过期） | 可执行计划 |

**根目录只允许**：本 `README.md`、分类子目录、`CI.md`。
